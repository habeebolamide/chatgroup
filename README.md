The ChatGroup API is a RESTful web service built with Node.js API where users in a chat group can receive and send messages

## Features

- Create Group
- Join Group
- Send Messages


## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose

## Getting Started
nodemon index.js

### Prerequisites

- Node.js (version 14.20.1)
- MongoDB (version 5.5.0)
